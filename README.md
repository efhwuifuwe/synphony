   Use the `cd` command followed by the path of the folder where you extracted Synphony. For example, type:

   Run the following command to install the required packages:
   ```
   npm install
   ```

6. **Start the Server**  
   To start the server, use the command:  
   ```
   npm start
   ```

7. **Access Your Server**  
   Open your web browser and type `http://localhost:3000` to view your running server.
